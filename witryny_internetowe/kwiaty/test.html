Sposoby wyświetlania elementów

Atrybut display pozwala określić , jakdany element ma być wyświtlony na stronie:

selektor(display: sposób;)
  • block - element wyświetlany jest jako element blokowy z odstepami u góry i od dołu
  • inline - element wyświetlany jest w lini z innymi elementami
  • none - element nie będzie wyświetlany
  • list-item - element wyświtlany jako element wykazu listy
